作物本质上也是方块, 只不过有着不同的模型大小且无碰撞体积. 所以实现起来也不难.
```java
public class BlockCornCrop extends BlockCrops {  
    public BlockCornCrop() {  
        setRegistryName("corn_crop");  
        setTranslationKey(Testmod.MOD_ID + ".corn_crop");  
        setCreativeTab(ItemBase.TEST_MOD_TAB);  
    }  
    @Override  
    protected Item getSeed() {  
        return ModItemList.CORN_SEED;  
    }  
    @Override  
    protected Item getCrop() {  
        return ModItemList.CORN;  
    }
}
```

其中重写的方法`getSeed()`和`getCrop()`就是在左键收获作物时会掉落的物品.


## 种子
```java
public class ItemCornSeed extends ItemSeeds implements IModelRegister {  
    public ItemCornSeed() {  
        super(ModBlockList.BLOCK_CORN_CROP, Blocks.DIRT);  
        setRegistryName("corn_seed");  
        setTranslationKey(Testmod.MOD_ID + ".corn_seed");  
        setCreativeTab(ItemBase.TEST_MOD_TAB);  
        ModItemList.items.add(this);  
    }  
    @Override  
    public void registerModels() {  
        Testmod.proxy.registerItemModel(this, 0, "inventory");  
    }
}
```

很常规的内容, 额外需要做的是调用超类构造函数. 告诉Forge这个种子是什么属于什么作物的以及可以种在什么方块上.
